# Learning Rust

A comprehensive collection of Rust learning materials, following a structured curriculum.

## Topics Covered

1. **Variables** - Mutability, constants, and shadowing.
2. **Numbers & Binary System** - Integer types, floating point, and literals.
3. **Chars, Bools & Unit Types** - Scalar types and the unit type `()`.
4. **Statements & Expressions** - Understanding the difference between instructions and values.
5. **Functions** - Parameters, return types, and implicit returns.
6. **Ownership** - Memory management, move semantics, and cloning.
7. **Borrowing** - References, mutability rules, and borrowing concepts.
8. **String vs. &str** - Owned strings vs string slices.
9. **Slices** - References to a contiguous sequence of elements.
10. **Tuples** - Grouping values of different types.
11. **Structs** - Custom data structures.
12. **Enums** - Types that can be one of several variants.
13. **The "Option" Enum** - Handling the presence or absence of a value.
14. **Flow Control** - `if`, `else`, `loop`, `while`, and `for`.
15. **Pattern Match** - Using `match` and `if let`.
16. **Methods & Associated Functions** - Implementation blocks and `self`.
17. **Generics** - Abstracting over types.
18. **Traits** - Defining shared behavior.
19. **Trait Objects** - Dynamic dispatch with `dyn Trait`.
20. **Associated Types** - Placeholders in traits for concrete types.
21. **String (Collection)** - Growing, shrinking, and manipulating UTF-8 strings.
22. **Vectors** - Resizable arrays for storing multiple values.
23. **HashMaps** - Storing key-value pairs.
