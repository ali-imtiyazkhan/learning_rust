# Learning Rust

A comprehensive collection of Rust learning materials, following a structured curriculum.

## Topics Covered

1.  **Variables** - Mutability, constants, and shadowing.
2.  **Numbers & Binary System** - Integer types, floating point, and literals (Hex, Octal, Binary).
3.  **Chars, Bools & Unit Types** - Scalar types and the special unit type `()`.
4.  **Statements & Expressions** - Understanding the difference between actions and values.
5.  **Functions** - Parameters, return types, and implicit vs explicit returns.
6.  **Ownership** - Memory management, move semantics, and cloning.
7.  **Borrowing** - References (`&`), mutability rules (`&mut`), and borrowing limits.
8.  **String vs. &str** - Owned `String` vs. string slices `&str`.
9.  **Slices** - References to a contiguous sequence of elements in a collection.
10. **Tuples** - Grouping multiple types. Includes **Partial Move** concepts.
11. **Structs** - Custom data structures. Includes **Partial Move** and area calculation examples.
12. **Enums** - Defining types with multiple variants.
13. **The "Option" Enum** - Rust's safe alternative to null values.
14. **Flow Control** - `if/else` expressions and loops (`loop`, `while`, `for`).
15. **Pattern Match** - Using the `match` operator and `if let` for concise control flow.
16. **Methods & Associated Functions** - Defining functionality within `impl` blocks.
17. **Generics** - Creating flexible, reusable code with type parameters.
18. **Traits** - Defining shared behavior across different types (similar to interfaces).
19. **Trait Objects** - Dynamic dispatch for using different types that implement the same trait.
20. **Associated Types** - Placeholder types within traits for more flexible designs.
21. **String Collection** - Working with the heap-allocated `String` collection.
22. **Vectors** - Growable arrays stored on the heap.
23. **HashMaps** - Storing key-value pairs for fast lookup.
24. **Type Coercion** - Automatic type conversions (like Deref coercion).
25. **From & Into** - Idiomatic type conversion traits.
26. **panic!** - Handling unrecoverable errors.
27. **Result & Error Handling** - Managing recoverable errors with `Result<T, E>` and the `?` operator.
28. **Cargo, Crates & Modules** - Organizing code into packages, crates, and modules for scope and privacy.
